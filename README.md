# 📈 Analisador de Criptomoedas em Python

Este projeto consiste em uma **pipeline de dados automatizada** desenvolvida em Python. O objetivo é realizar a coleta de preços de criptomoedas em tempo real através da API da CoinGecko, armazenar os dados de forma persistente em um arquivo CSV e gerar visualizações gráficas para monitoramento de mercado.

## 🛠 Tecnologias Utilizadas
- **Python**: Linguagem principal.
- **Pandas**: Estruturação e manipulação de séries temporais.
- **Requests**: Consumo de APIs externas.
- **Matplotlib**: Geração de gráficos para análise visual.

### 1. Pré-requisitos
Certifique-se de ter o Python instalado em sua máquina.
